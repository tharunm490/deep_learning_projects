# Customer Churn Prediction using ANN

This project is a **basic Deep Learning implementation** of a **Customer Churn Prediction system** using an **Artificial Neural Network (ANN)** built with **TensorFlow and Keras**.

The goal of this project is to predict whether a customer is likely to **leave (churn)** or **stay** based on customer-related features.
